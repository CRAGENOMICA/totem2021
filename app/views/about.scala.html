Your organism of interest is not available?

TOTEM has been designed as a modular tool. Potentially any experiment containing tissue-specific data (or identifying a set of tissue-specific genes) can be added without major code changes. If you would like TOTEM to incorporate a specific experiment/organism, we encourage you to send us a brief description of your proposal, the paper describing the experiment and a draft drawing (preferentially in a SVG format).

Disclaimer

The generated image is merely a visual representation and it may not depict all tissues examined in the cited experiment. It is the barplot which contains the actual enrichment scores for all tissues of the experiment. With TOTEM we only try to facilitate the comparison with published spatio-temporal expression data. Our criteria to define tissue-enriched genes may not satisfy all necessities, thus conclusions should be drawn at the risk of the users.
